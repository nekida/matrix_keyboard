# matrix_keyboard
Use matrix keyboard on NUCLEO-F031K6 board with MCU stm32f031x6


Работа с матричной клавиатурой и проверкой введенного с нее пароля. Вывод результата ввода в UART1 на скорости 38400. Самое интересное происходит в /Src/main.c. Детали проекта с распиновкой и инициализацией в matrix_keyboard.pdf и matrix_keyboard.txt
